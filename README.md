# 📚 App de Révision - Fin de la WWII et Nouvel Ordre Mondial

Une application web complète pour réviser le chapitre d'Histoire sur la fin de la Seconde Guerre mondiale et les débuts du nouvel ordre mondial (1944-1947).

## 🎯 Fonctionnalités

✅ **🎴 Flashcards** - 25 cartes pour mémoriser les dates et événements clés  
✅ **❓ Quiz Interactif** - 10 questions avec correction automatique  
✅ **📝 Sujets de Dissertation** - 3 sujets avec plans structurés  
✅ **✏️ Entraînement à la Rédaction** - 3 exercices avec feedback automatique  
✅ **🧠 Mémorisation Espacée** - 15 termes avec système de révision intelligent  
✅ **🔗 Partage avec QR Code** - Partagez l'app et votre progression  

## 🚀 Déploiement sur GitHub Pages

### Étape 1 : Créer un repository GitHub

1. Allez sur [github.com](https://github.com)
2. Cliquez sur **"+"** → **"New repository"**
3. Donnez un nom : `app-revision-histoire` (ou autre)
4. Sélectionnez **"Public"**
5. Cliquez sur **"Create repository"**

### Étape 2 : Uploader les fichiers

**Option A : Via GitHub (plus facile)**
1. Dans votre nouveau repository, cliquez sur **"Add file"** → **"Upload files"**
2. Drag & drop les fichiers :
   - `app_revision.html`
   - `fin_de_la_seconde_guerre_mondiale_et_nouvel_ordre_mondial.md`
   - `README.md`
3. Cliquez sur **"Commit changes"**

**Option B : Via Git en ligne de commande**
```bash
git clone https://github.com/votreusername/app-revision-histoire.git
cd app-revision-histoire
# Copiez les fichiers ici
git add .
git commit -m "Initial commit: app de révision d'histoire"
git push origin main
```

### Étape 3 : Activer GitHub Pages

1. Allez dans **Settings** de votre repository
2. Dans la barre de gauche, cliquez sur **"Pages"**
3. Sous **"Source"**, sélectionnez **"Deploy from a branch"**
4. Sélectionnez **"main"** comme branche
5. Sélectionnez **"/ (root)"** comme dossier
6. Cliquez sur **"Save"**

### Étape 4 : Attendez le déploiement

- GitHub va créer l'URL automatiquement
- Vous verrez un message : "Your site is live at https://username.github.io/app-revision-histoire/"
- Ça peut prendre 1-2 minutes
